User Stories

- User can view home page with sample songs
- User can got to login page and login with their spotify account
- User can logout
- User can view further info such as about and contact from the footer
- User can see the home page with Featured songs, Albums of the summer etc.
- User can play, Add songs to playlists and add to their Liked songs
- User can Navigate to all pages from sidebar, Search songs, go to library,
  view genres, view their liked songs and playlists
- User can view their songs, albums, artists, playlists, genres and podcasts
 from the library page
- User can play and add songs from all pages
- User can view, create and delete playlists and liked songs
 from corresponding pages (CRUD)
